# 🏨 Chalé Hotel

Projeto desenvolvido durante o curso **Desenvolvimento Web Completo** da Udemy, com o objetivo de praticar conceitos fundamentais de HTML5 e CSS3 por meio da criação de um site institucional para um hotel fictício.

## 📷 Demonstração

**URL** -> <a href="https://maicondante.github.io/Projeto_Chale_Hotel/">Projeto Chalé Hotel</a>

---

## 📖 Sobre o Projeto

O Chalé Hotel é um site fictício desenvolvido para simular a presença digital de um hotel, apresentando informações sobre hospedagem, serviços e reservas.

O projeto foi criado com foco na aplicação de boas práticas de desenvolvimento web utilizando HTML5 e CSS3, explorando a organização semântica do conteúdo e a construção de uma interface visual agradável para o usuário.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Folha de estilo externa (`style.css`)
* Estrutura semântica com HTML5

---

## 📐 Estrutura Semântica

Durante o desenvolvimento foram utilizadas tags semânticas para melhorar a organização e legibilidade do código:

* `<header>` — Cabeçalho com identidade visual do hotel
* `<nav>` — Menu de navegação
* `<main>` — Conteúdo principal
* `<section>` — Seções informativas
* `<footer>` — Informações complementares e rodapé

Essa abordagem contribui para uma melhor organização do projeto e segue as boas práticas do desenvolvimento web moderno.

---

## 🌐 Conteúdo do Site

O site apresenta informações relacionadas ao hotel e seus serviços, incluindo:

* 🏨 Apresentação do Chalé Hotel
* 🛏️ Informações sobre hospedagem
* ⭐ Descrição dos serviços oferecidos
* 📸 Imagens ilustrativas do ambiente
* 📅 Área de reservas
* 📞 Informações de contato

A estrutura foi desenvolvida para proporcionar uma navegação simples, intuitiva e organizada.

---

## 🎯 Objetivos de Aprendizagem

Durante a construção deste projeto foram praticados conceitos como:

* Estruturação de páginas com HTML5
* Utilização de tags semânticas
* Estilização de páginas com CSS3
* Organização de conteúdo institucional
* Criação de layouts estáticos para negócios
